# Package ucdparser

Downloading or updating:

``` bash
go get -u github.com/adnsv/ucdparser
```

In your code:

``` go
import "github.com/adnsv/ucdparser"
```

## Overview

Package ucdparser is a simple generic parser for
unicode.org reference tables.
